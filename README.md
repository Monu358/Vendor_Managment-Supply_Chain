# Vendor_Managment-Supply_Chain
This project addresses the challenge of evaluating vendor performance across key supply chain metrics: - Delivery punctuality (on-time vs early deliveries) - Product quality (rejection rates) - Freight and associated costs.
## Problem Statement
This project addresses the challenge of evaluating vendor performance across key supply chain metrics:
 Delivery punctuality (on-time vs early deliveries)
 Product quality (rejection rates)
 Freight and associated costs

The goal is to identify inefficiencies, reduce costs, and improve quality through data-driven vendor management strategies.

## Tools Used
  SQL Server: For data extraction, transformation, and KPI calculations
  Power BI  : For interactive dashboards and trend analysis
  Canva     : For visual storytelling and summary infographics
